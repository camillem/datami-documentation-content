# Quickstart

---

## Project setup

```bash
nvm use
npm install
cp example.env .env
```

---

### Compiles and hot-reloads for development

```bash
npm run serve
```

By default Gitribute will run on `https://localhost:8080`

---

### Compiles and minifies for production

```bash
npm run build
```

---

### Run your unit tests

```bash
npm run test:unit
```

---

### Lints and fixes files

```bash
npm run lint
```
